# Voluptia V143 — recherche avancée + Libre ce soir

## Modifications principales

- Ajout d’une barre de recherche latérale inspirée de la capture envoyée.
- Ajout des boutons On/Off :
  - Proche de mon profil ;
  - Avec photos ;
  - Libre ce soir.
- Ajout des sections pliables :
  - Principaux critères ;
  - Localisation ;
  - Physique ;
  - Autres critères ;
  - Recherches mémorisées.
- Ajout de deux blocs de recherche :
  - Je recherche ;
  - Qui recherchent.
- Ajout des filtres :
  - Tous les couples ;
  - Toutes les femmes ;
  - Tous les hommes ;
  - Tous les trios ;
  - Tous les groupes ;
  - Femme hétéro, femme bi, femme lesbienne ;
  - Homme hétéro, homme bi, homme gay ;
  - Couple, trio et groupe avec variantes bi/mixte.
- Suppression du critère Premium dans la nouvelle recherche avancée.
- Ajout des recherches mémorisées côté navigateur.
- Ajout du statut personnel Libre ce soir dans Mon espace > Mon profil > Préférences.
- Synchronisation backend du statut Libre ce soir avec le profil.
- Les profils marqués Libre ce soir peuvent maintenant ressortir avec le filtre correspondant.
